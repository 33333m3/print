Answer in numbers / INPUT

A program that address various addition problems using "RND"(random).
Use "LET" to store numbers on A to Z.
Let's "RUN" several times!
In order to store the numbers using the keyboard, use "INPUT" command.
Infront of the "?", type a number with the keyboard and press ENTER key.
The number put in C is memorized!
For checking answers, "IF" is used to judge.
Let's use "ELSE" when you want to show a miss.
It can also appear with a question text.
Try it!
